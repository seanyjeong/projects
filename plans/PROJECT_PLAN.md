## 4. 기술 스택

```
Frontend
├── Next.js 15 (App Router)
├── TypeScript
├── TailwindCSS
├── Shadcn/ui
└── 모바일 우선 반응형

Backend
├── NestJS
├── TypeScript
├── Swagger
└── univjungsi 계산 엔진

Database
├── MySQL (기존 서버)
└── Prisma ORM

Infrastructure
├── Vercel (Frontend)
└── 기존 서버 (Backend + MySQL)

Authentication
├── NextAuth.js
└── 카카오/네이버 소셜

Payment
└── 토스페이먼츠
```
