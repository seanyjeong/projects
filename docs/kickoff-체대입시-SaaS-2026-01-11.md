## 3. 주요 결정 사항 (CEO 승인)

### CEO 답변
| 질문 | 답변 |
|------|------|
| 기존 코드 | `maxjungsi222/` (프로덕션) + `univjungsi/` (리팩토링) |
| 지원 범위 | **정시 + 수시** (MVP는 정시 우선) |
| 타겟 사용자 | **수험생 개인 (B2C)** |
| 서비스명 | 미정 |
| 인프라 예산 | **$20 이하/월** |
| 출시 목표 | **최대한 빠르게** |

### 기술 스택 (확정)
- **Frontend:** Next.js 15 + TypeScript + TailwindCSS (모바일 우선)
- **Backend:** **NestJS** (CEO 결정 - 리팩토링 기준)
- **DB:** **MySQL + Prisma** (기존 서버 활용, $0)
- **인프라:** Vercel (프론트) + 기존 서버 (백엔드)
- **계산 엔진:** univjungsi/src/lib/calculator 재사용 (31개 테스트 통과)

### 도메인 지식
- **정시 가/나/다군**: 각 군별 1개 대학만 지원 가능 (최대 3개)
- 체대는 대학마다 군이 다름 → 전략적 조합 중요
- 가군(1월 초), 나군(1월 중), 다군(1월 하)

### 수익 모델 (민수 제안)
- Freemium 모델
- 무료: 기본 계산기
- 프리미엄: 합격예측, AI 추천, 상세 분석

### 디자인 방향 (CEO 지시 + 디자인팀 합의)
- **모바일 우선 (90%+ 모바일 사용자)** - 수험생은 폰으로 사용
- 미니멀 기본 + 포인트 컬러
- 다크모드 지원
- 랜딩페이지는 글래스모피즘
- PC는 반응형으로 대응
